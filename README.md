Habit Tracker Web Application

Description
A full-stack Habit Tracker web application built with Node.js, Express, MongoDB, and vanilla JavaScript.

Features
- Create, view, update, and delete habits
- Production-ready web interface
- MongoDB database
- Environment variable configuration

Technologies
- Node.js
- Express
- MongoDB
- HTML, CSS, JavaScript

Setup
1. Clone repository
2. Install dependencies:
   npm install
3. Create .env file with:
   PORT
   MONGO_URI
4. Run:
   node server.js
